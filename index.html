Getting Started with Overload Engine

Welcome to Overload Engine! This guide will help you set up your first mod.

Installation

  1. Download the latest release from the Downloads page
  2. Extract the ZIP file to your preferred location
  3. Run OverloadEngine.exe to launch

Your First Mod

Create a new folder in the mods/ directory. Each mod needs:

  • mod.json — Mod metadata
  • scripts/ — HScript files
  • assets/ — Images, sounds, charts

Project Structure

mods/your-mod/ mod.json scripts/ main.hsc assets/ images/ sounds/ charts/

Deep State Injection

Deep State Injection allows you to load any game state (Main Menu, Story Mode, Freeplay) inside an editor overlay for live tweaking.

How It Works

The engine captures the current state and renders it within the editor context, allowing real-time modifications.

Usage

Open the editor with F12, then use the State dropdown to inject any game state into the current view.

API

Overload.injectState('FreeplayState'); Overload.injectState('StoryMenuState');

Ultimate UI Builder

The drag-and-drop interface lets you create complex UIs without writing code.

Features

  • Visual layout editor
  • Live HScript code preview
  • Asset spawning from the browser
  • Snap-to-grid alignment
  • Layer management

Quick Start

  1. Press Tab to open the UI Builder
  2. Drag elements from the toolbox
  3. Configure properties in the inspector
  4. Click 'Export HScript' to generate code

HScript API Reference

Overload Engine extends the base HScript with powerful APIs.

Core Functions

// Spawn a sprite var spr = Overload.createSprite(x, y, 'path/to/image'); // Add tween Overload.tween(spr, {x: 500, alpha: 0.5}, 1.0, 'easeOutQuad'); // Play sound Overload.playSound('path/to/sound'); // Register callback Overload.onBeat(function() { // runs every beat });

Editor API

Overload.editor.addPanel('My Panel', myCallback); Overload.editor.registerTool('brush', brushCallback);

Scripting Guide

Learn how to write scripts for Overload Engine.

Live Script Console

Press ` to open the Live Script Console. Write HScript and click 'Apply' to see changes without restarting.

Script Lifecycle

function onCreate() { // Called when the script loads } function onUpdate(elapsed) { // Called every frame } function onDestroy() { // Called when the script unloads }

Error Handling

Overload Engine provides smart error messages. If a script fails, you'll see a crash substate explaining exactly what went wrong.
